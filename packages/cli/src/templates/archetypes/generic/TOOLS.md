# TOOLS.md — Ferramentas (Generico)

## Built-in

- **File ops:** read, write, edit, glob, grep em `{{workingDir}}`.
- **Shell:** comandos gerais.
- **Git:** status, diff, log, commit.

## Voz

- Whisper (Groq por padrao, OpenAI opcional).

## Integracoes recomendadas

- **Obsidian** — opcional, se usa um vault.
- **GitHub CLI (`gh`)** — opcional, se usa repositorios.

## MCP servers sugeridos

- `context7` — docs oficiais quando pesquisa algo tecnico.

## Como habilitar um MCP

Edite `~/.forgeclaw/forgeclaw.config.json` -> `mcpServers`.
