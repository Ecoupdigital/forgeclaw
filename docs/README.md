## Ordem sugerida de leitura

1. [getting-started.md](./getting-started.md) — voce instala e manda a primeira mensagem
2. [archetypes.md](./archetypes.md) — quando quiser entender o que escolheu (ou trocar)
3. [dashboard-tour.md](./dashboard-tour.md) — ja com o ForgeClaw rodando, para navegar no dashboard
4. [harness-guide.md](./harness-guide.md) — quando quiser personalizar o comportamento
5. [agents.md](./agents.md) — quando quiser ter especialistas por topico
6. [crons.md](./crons.md) — quando quiser automacao proativa

Duvidas pontuais: [troubleshooting.md](./troubleshooting.md) e [faq.md](./faq.md) sao seus amigos.
