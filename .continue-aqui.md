### Bugs conhecidos / Pendências
1. **Dashboard usa mock data como fallback** — API routes integradas mas algumas podem falhar se core não inicializou
2. **Cron execution** — jobs disparam mas resultado depende do Claude CLI (consome contexto da conta Max)
3. **Auth no grupo** — user 1087968824 (GroupAnonymousBot) é bloqueado pelo middleware de auth; considerar whitelist de grupo também
4. **Streaming throttle** — edits a cada 500ms, mas mensagens muito longas podem exceder 4096 chars do Telegram
5. **systemd não configurado** — bot roda via nohup, não sobrevive reboot
